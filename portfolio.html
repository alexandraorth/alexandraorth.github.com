Fall 2013

Stop & Frisk: Ammunition for Gentrification?

This investigation explored the relationship between changing gentrification indexes and the increased number of 'stop and frisks' in New York City. Raw data was provided from NYC Open Data and was then cleaned, indexed and entered into a database. Both maps are written in d3.js.

July & August 2013

AppFirst Pod Server Architecture

This graphing system shows the interconnections between servers and applications within an AppFirst pod. The mapping is based on IP address and nodes can be gathered and separated into their components. Built on Sinatra with a Mongo database, the data visualization was written in d3.js.

June & July 2013

AppFirst Automated Alerting

This alerting system was based on Etsys' Skyline and detects single point anomalies in data streams provided by AppFirst collectors. Using running averages and the grubbs test, alerts will be triggered without the need to configures individual thresholds for each server or application. The app is built on the Rails framework with a Mongo database. D3.js was used for all graphing.
